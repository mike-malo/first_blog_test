第一章

表达式与命名：

原始表达式：

例如：27，False，4.27

每一个都是一个基本表达式，因为计算它们不需要计算。

算术表达式：

Python附带了四个基本的算术运算符：+, -, *, /

** ：这是求幂运算符, 例如2的平方在python中就是2**2

// ：它将对除法的结果进行四舍五入并返回一个整数。

% ：它将计算除法的正余数。

注意：/ 将返回一个浮点数，例如：3/3将返回1.而不是1，另外//和%将返回整数

命名：

在Python中，与任何其他编程语言一样，可以为变量名赋值。这允许在代码执行时
重用和更新值，从而为您提供更大的灵活性和功能性。
例如：
->c = 5 + 2
->c
7

函数（FUNCTIONS）：

什么是函数？

和代数一样，函数是接受一个或多个输入，用这个输入做一些事情，然后返回一个输出。

示例：

f() = 5

此函数始终返回5，在python中实现：

def f():
return 5

我们可以反复使用这个函数：

y = f()

请注意，将y赋给f（）会将其赋给调用f（）的结果，而不是函数本身。因此，将y更改为6不会更改f（）的返回值。

函数的剖析

在我们分析一个函数的解剖结构之前，让我们先看看另一个函数：

g(x) = x + 2

在数学中，输入x就会返回x+2，在python中实现：

def g(x):
return x + 2

我们来分解这个函数：

有一个函数叫做‘ g ’，它接受一个输入‘ x ’，‘ g ’ 返回 ‘ x + 2 ’

请注意，一个函数可以有多行，但只能有一条返回语句。

def h(x):
y = x + 2
return y

这个函数的作用与g完全相同，但它首先将x+2赋值给一个新变量，然后再返回该新变量。

内置函数

python附带了一些函数，这里例举一部分

->print(‘Hello!’) #打印括号内的值，不带引号。
Hello!
->print(print(‘Hello!’)) #请参阅下一节，了解如何对其进行。
Hello! # “print”函数打印输入
None # …但请注意，它返回None，这是nothing的表示。
->min(2, 5) # 返回两个或多个数字中的最小值，用逗号分隔。
2
->max(2, 5) # 返回最多两个或多个数字，用逗号分隔。
5

求值（Evaluation）

函数的求值方式是理解其工作原理的关键。本质上，Python在计算运算符之前会尝试计算每个操作数。在h（x）中，h是运算符（operator），x是操作数（operand）。

在本例中，我们将使用函数add和mul，这样您就可以看到PEMDAS风格的操作。这些函数在operator软件包中，您将在下周了解这些函数。现在，请忽略下面示例中的import语句。

->from operator import mul, add
->add(5, mul(add(5, 2), 4))
33

那么这里发生了什么？

  • Python尝试计算5，然后是mul(add(5, 2), 4)。
    1.在计算第二个表达式*mul(add(5, 2), 4)*时，Python尝试计算add(5, 2)和4。

     在计算第一个表达式*add(5, 2)*时，Python尝试计算5和2。
 这些已经被计算过了，所以最里面的add函数执行并返回7。

    2.现在，mul()函数的参数变为7和4，因此mul()函数执行并返回28。

  • 最后，最外层的add()函数的参数变成了5和28，因此该函数执行并返回33。

环境图

既然我们知道了如何开始基本的编程，我们还应该研究如何可视化我们的代码。在很大程度上，这是相对困难的，因为项目变得太大，无法可视化。然而，当第一次学习如何编程时，可视化代码及其运行方式非常有用。
